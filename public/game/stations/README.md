# Station monsters

Monster sprites from the [Ninja Adventure Asset Pack](https://pixel-boy.itch.io/ninja-adventure-asset-pack)
by Pixel-boy & AAA — CC0 1.0. Same sheet layout as the buddies: 16×16 frames,
4 direction columns (down, up, left, right) × 4 animation-frame rows.

Cute set (Public collections): racoon, panda, owl, butterfly, mouse, axolot, mole, bear.
Scary set (System levels, in rising order): slime, mushroom, kappa, lizard,
snake, cyclope, skull, flam, dragon, trex.
